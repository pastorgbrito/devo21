# **Explicación del archivo JSON** 

# **cabecera**

titulo: Este es el título general de todo el estudio  
descripcion: Una breve descripción del estudio  
tematica: La temática en la que se clasifica el estudio  
fecha\_creacion: Fecha en que se crea el estudio yyyy-mm-dd (ej. "2026-02-27")
autor: Nombre de persona que creó el estudio  
duracion\_dias: tiempo de duración del estudio expresado en días

# **cuerpo_del_estudio**

dia: Numero del dia correspondiente al estudio (Ejemplo: dia 1; dia 2; etc.)  
titulo\_dia: Es el título del estudio del día  
lectura\_base: Es el texto a estudiar ese día  

## **observacion: Primera sección del método inductivo de estudio bíblico**

* ayuda\_ia: Texto exclusivo para dar contexto al agente IA. Para que la IA sepa el resultado que debes esperar de parte del usuario. La IA debe redactar las preguntas necesarias, en base a esta ayuda, para que el usuario observe lo correcto.   
* resumen: Un resumen de la observación del texto

## **interpretacion**

## **aplicacion**

* contexto\_real: Explica el contexto histórico/geográfico del pasaje de forma clara, corta y precisa para entender el escenario real.  
* contexto\_actual: Analogía moderna del principio bíblico aprendido.   
* examen\_personal: Pregunta de introspección sin culpa.  
* accion\_24h: Un micro-hábito práctico y sencillo.  
* oracion\_de\_sellado: Una pequeña oración para sellar la enseñanza
