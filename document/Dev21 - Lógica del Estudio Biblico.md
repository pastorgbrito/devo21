# **Lógica del Estudio Biblico**

# **Estructura y Disparador de Inicio**

* **Días Contables**: la duración del Estudio.  
    
* **Botón de Inicio**: Disparador obligatorio para visualizar el contenido y registrar la hora de inicio (puntualidad).  
    
* **Entrega Secuencial**: El contenido se libera paso a paso tras presionar el botón, impidiendo que el usuario vea la oración final antes de completar las etapas previas.  
    
* **Restricción de Avance**: Prohibido llevar a cabo estudios de días futuros.

# **Reglas de Recuperación (Ponerse al día)**

**Días de Semana (L-V)**: Máximo un estudio por día, del mismo estudio, ya que el usuario puede estar realizando varios al mismo tiempo. Pero del mismo estudio, no se permite realizar varios días de estudios  de un mismo estudio. Los estudios solo se podrán realizar de lunes a viernes. El usuario, por configuración, podrá definir si lo hace de lunes a viernes solamente, o toda la semana, incluyendo sábado y domingo.

**Fines de Semana (S-D)**: Únicos días habilitados para reposición (máximo 1 por día).

**Cierre del Desafío**: Al cumplir los días de duración de cada estudio \+ prórroga de 4 días de gracia.

# **Reglas de Navegacion** 

El estudio se mostrará al usuario por secciones:

1. **Primera Sección**: La pantalla de inicio del estudio, donde aparecen los días de estudio, y se ven los que han realizado y los que le faltan. También estará el botón de inicio para sincronizar la hora de arranque del estudio, para llevar la estadística de puntualidad.  
2. **Segunda Sección**: Acá estará el lector bíblico, donde aparece el texto del día. Y un botón para que, al terminar, podamos avanzar a la siguiente sección.  
3. **Tercera Sección**: Es la sección de **Observación**, y es en esta pantalla donde el usuario interactúa con el agente IA,  para que lo guíe a observar bien el texto. Es un chat como el de WhatsApp, pero interactuando con el agente. Un botón para que, al terminar, podamos avanzar a la siguiente sección.  
4. **Cuarta Sección**: Es la sección de **Interpretación.** Es igual que la anterior, solo que la misión del agente es distinta: llevar al usuario a interpretar el texto bíblico. Un botón para que, al terminar, podamos avanzar a la siguiente sección.  
5. **Quita Sección:** Es la sección de **Aplicación.** Es una ventana donde se muestran los datos que vienen en el JSON, para ese día. Y el botón de finalizar estudio.
